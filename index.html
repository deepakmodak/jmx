Welcome to JMX !

'JMX' is basic "hello world" application showcasing how to use it along with jconsole.

Contents :

  • 1 java interface SampleMBean
  • 2 java classes Sample TestAgent
  • ant used to compile java files and build jar

Steps to run after downloading

  1. ant
  2. cd dist/
  3. java -jar jmx.jar
